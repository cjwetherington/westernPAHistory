# Western PA History Child Theme Plugin

## Version
1.0 for OJS 3.3

## Purpose
A Health Sciences child theme plugin for the Western PA History journal. It includes
custom template and style overrides to ensure that the theme conforms to accessibility
guidelines.

## Dependencies
Requires and extends OJS' Health Sciences theme.

## Installing
- Place folder in the site's `plugins/themes` subfolder and set appropriate permissions.  
- Enable the plugin via Website > Plugins > Installed Plugins.
- Activate the theme via the dropdown at Website > Appearance > Theme.
